## 대시보드

메인 대시보드는 모든 핵심 지표를 한눈에 보여주는 반응형 카드 기반 레이아웃을 사용합니다.

### KPI 카드

4개의 최상위 KPI 카드가 실시간 지표를 표시합니다:

| 카드 | 지표 | 설명 |
|---|---|---|
| **총 호출 수** | 건수 | 추적된 총 함수 실행 횟수 |
| **평균 지연시간** | 밀리초 | 모든 함수의 평균 실행 시간 |
| **에러율** | 백분율 | 실패 대비 전체 실행 비율 |
| **캐시 히트율** | 백분율 | 시맨틱 캐시 히트 비율 |

### 실행 타임라인

시간별 함수 실행을 보여주는 실시간 차트:

- 성공/에러/캐시 분류가 적용된 누적 영역 차트
- 설정 가능한 시간 범위 (1시간, 6시간, 24시간, 7일, 30일)
- 호버 시 타임스탬프별 상세 내역 표시
- 설정 가능한 간격으로 자동 새로고침

### 에러 분포

에러 유형의 도넛 차트:

- 에러 코드 또는 예외 유형별 그룹화
- 클릭하면 해당 에러 유형으로 실행 목록 필터링
- 심각도별 색상 구분

### 위젯 커스터마이징

드래그 앤 드롭으로 대시보드 레이아웃을 커스터마이징합니다. 위젯 크기를 조절하고 사용자별 레이아웃을 저장할 수 있습니다.

## 트레이스 워터폴

트레이스 워터폴은 분산 호출 체인을 인터랙티브 다이어그램으로 시각화합니다.

### 워터폴 읽기

```
pipeline()                    ████████████████████████░░░░░ 450ms
├── embed()                   ██████████░░░░░░░░░░░░░░░░░░  180ms
├── search()                  ░░░░░░░░░░████████░░░░░░░░░░  150ms
└── format_result()           ░░░░░░░░░░░░░░░░░░██░░░░░░░░   20ms
```

- **수평 막대**는 스팬 지속 시간을 나타냅니다
- **중첩**은 부모-자식 관계를 보여줍니다
- **색상**은 상태를 나타냅니다:
  - 초록색 = 성공
  - 빨간색 = 에러
  - 파란색 = 캐시 히트
  - 회색 = 진행 중

### 스팬 상세 정보

스팬을 클릭하면 확인할 수 있는 정보:

- 함수 이름 및 모듈 경로
- 입력 파라미터 및 반환값
- 에러 상세 정보 및 스택 트레이스 (실패한 경우)
- 소요 시간 분석 및 캐시 히트 여부

### 트레이스 필터링

- 함수 이름, 상태, 소요 시간 범위로 필터링
- 타임스탬프, 소요 시간, 에러 횟수로 정렬
- Trace ID로 트레이스 검색

## AI Healer

AI Healer는 에러를 자동으로 진단하고 코드 수정을 제안합니다.

### 에러 브라우저

다음 정보와 함께 모든 최근 에러를 탐색:

- 에러 유형 및 메시지
- 영향을 받은 함수 및 모듈
- 타임스탬프 및 발생 빈도
- 스택 트레이스 미리보기

### 단건 진단

1. 목록에서 에러를 선택
2. **진단** 버튼을 클릭하여 AI Healer 실행
3. 진단 결과 확인:
   - 근본 원인 분석
   - diff 하이라이팅이 적용된 코드 수정 제안
   - 신뢰도 점수
4. **PR 생성**을 클릭하여 GitHub Pull Request 오픈

### 배치 진단

1. 여러 에러를 선택 (또는 **전체 선택**)
2. **배치 진단** 클릭
3. 요약 뷰에서 모든 진단 결과 검토
4. 승인된 수정에 대해 PR 생성

### 진단 이력

과거 진단과 결과를 확인할 수 있습니다 — 병합된 수정, 거부된 수정, 시간에 따른 성공률.

## 리플레이 테스트

AI 함수를 위한 시각적 회귀 테스트.

### 리플레이 실행

1. 사이드바에서 함수를 선택
2. 테스트 항목을 선택:
   - **Golden만** — 검증된 Golden Dataset 항목만
   - **전체** — 저장된 모든 실행
3. 비교 모드를 선택:
   - **정확 일치** — 문자 단위 비교
   - **시맨틱** — 설정 가능한 임계값을 사용한 임베딩 유사도
4. **리플레이 실행** 클릭

### 결과 보기

- **요약** — 통과/실패 건수, 성공률
- **상세 보기** — 예상 출력과 실제 출력의 나란히 비교
- **유사도 점수** — 시맨틱 비교의 코사인 유사도
- **Diff 보기** — 정확 일치 실패에 대한 문자 단위 diff

### Golden Dataset 관리

- 함수별 모든 Golden Dataset 항목 탐색
- 새로운 실행을 Golden 상태로 승격
- 오래된 항목 제거
- AI가 추천하는 승격 후보 확인

## 시맨틱 검색

모든 AI 함수 데이터에 대한 전문 및 벡터 검색.

### 검색 인터페이스

- 상단의 자연어 검색바
- 입력 시 실시간 결과 표시
- 함수 이름, 상태, 날짜 범위, 에러 코드로 필터링

### 검색 결과

결과에 포함되는 항목:

- **함수** — 매칭 함수 정의
- **실행** — 입출력 미리보기가 포함된 매칭 실행 로그
- **에러** — 스택 트레이스가 포함된 매칭 에러 로그
- **트레이스** — 워터폴 미리보기가 포함된 매칭 트레이스 체인

### RAG Q&A

**AI에게 질문하기**를 클릭하여 자연어 Q&A 사용:

- 코드베이스에 대한 질문을 입력
- 데이터에 기반한 컨텍스트 답변 제공
- 검증을 위한 소스 링크 포함

| 언어 | 코드 |
|---|---|
| 영어 | `en` |
| 일본어 | `ja` |
| 한국어 | `ko` |

설정 메뉴에서 언어를 전환할 수 있습니다. 모든 UI 요소, 라벨, 메시지가 번역되어 있습니다.
