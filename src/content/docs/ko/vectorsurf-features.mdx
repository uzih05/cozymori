## 대시보드

메인 대시보드는 커스터마이징 가능한 위젯 그리드 레이아웃을 사용합니다. **Edit Mode**를 활성화하여 위젯을 추가, 제거, 크기 조절, 순서 변경할 수 있습니다.

### 사용 가능한 위젯

| 위젯 | 크기 | 설명 |
|---|---|---|
| **KPI Overview** | S / M / L | 5개의 KPI 카드: 총 실행 수, 성공률, 에러 수, 캐시 히트율, 평균 소요시간. 각각 스파크라인과 이전 기간 대비 트렌드 표시. |
| **Execution Timeline** | M / L | 시간별 실행을 보여주는 막대/선 차트. 3개 시리즈: 성공(초록), 에러(빨강), 캐시 히트(파랑). 선택한 시간 범위에 따라 구간 크기 자동 조절. |
| **Cache Hit Rate** | S | 캐시 히트 비율을 보여주는 원형 게이지. 히트/전체 건수 표시. |
| **Error Rate** | S | 에러율을 큰 숫자로 표시. 색상 구분: 파랑 (5% 미만), 주황 (5–10%), 빨강 (10% 초과). |
| **Token Usage** | S / M | 총 토큰 수와 카테고리별 도넛 차트. |
| **Function Distribution** | M / L | 실행 횟수 기준 상위 10개 함수의 수평 막대 차트. |
| **Recent Errors** | M / L | 최신 에러의 스크롤 가능한 목록. 함수명, 에러 코드, 메시지, 상대 시간 표시. |
| **System Status** | S | Weaviate 연결 상태 (연결됨/끊김), 등록된 함수 수, 마지막 확인 시각. |
| **Suggest Overview** | M / L | 우선순위 배지가 적용된 상위 5개 AI 제안 (심각/높음/보통/낮음). 전체 Suggest 페이지로 링크. |

### 위젯 크기

- **S** (Small) — 1 컬럼
- **M** (Medium) — 2 컬럼
- **L** (Large) — 4 컬럼, 전체 너비

### 대시보드 컨트롤

- **시간 범위** — 프리셋(15분, 30분, 1시간, 3시간, 6시간, 12시간, 24시간, 3일, 7일) 또는 커스텀 날짜-시간 범위 선택. 모든 시간 기반 위젯이 함께 업데이트.
- **차트 채우기** — 선, 그라디언트, 솔리드 채우기 스타일 선택.
- **새로고침** — 모든 위젯 데이터 수동 새로고침.
- **Edit Mode** — 토글하여 위젯 추가/제거/크기 조절. 각 위젯에 S/M/L 크기 버튼과 제거 버튼 표시.

## Ask AI

RAG를 활용한 VectorWave 데이터에 대한 자연어 Q&A.

### 사용법

1. 사이드바에서 **Ask AI**로 이동
2. **함수 범위** 선택 — "All Functions" 또는 특정 함수를 선택하여 컨텍스트 제한
3. 텍스트 영역에 질문 입력 (예: "embed 함수가 왜 실패하나요?")
4. **Ask** 클릭

AI가 Weaviate에서 관련 함수 데이터, 실행 로그, 에러 컨텍스트를 검색한 후 답변을 생성합니다.

### 플랜별 제한

| 플랜 | 일일 AI 호출 | 저장된 응답 접근 |
|---|---|---|
| **Free** | 일 5회 | 24시간 후 잠금 |
| **Pro** | 무제한 | 전체 히스토리 접근 |

Free 플랜 사용자에게는 오늘 남은 호출 수를 보여주는 사용량 바가 표시됩니다. 일일 한도 도달 시 Ask 버튼이 비활성화되고 "Daily limit reached. Upgrade to Pro for unlimited access." 메시지가 표시됩니다.

### 저장 및 북마크

Ask AI의 응답은 자동으로 **AI History**에 저장됩니다. 결과의 북마크 아이콘을 클릭하여 중요한 응답을 북마크할 수 있습니다.

## AI History (저장된 응답)

Ask AI와 Healer의 모든 AI 응답을 탐색하고 관리합니다.

### 필터링

- **소스 필터** — 전체, Ask AI만, 또는 Healer만 표시
- **북마크** — 북마크된 응답만 표시하도록 토글
- **검색** — 질문과 답변에 대한 전문 검색

### 응답 카드

각 카드에 소스 아이콘, 질문 미리보기, 소스 타입 배지, 함수명 (범위 지정된 경우), 타임스탬프, 북마크 상태가 표시됩니다. 클릭하면 전체 답변이 펼쳐집니다.

### 작업

- **북마크 / 북마크 해제** — 응답의 북마크 상태 토글
- **삭제** — 응답을 영구 삭제 (확인 대화상자 포함)

### Free 플랜 제한

Free 플랜에서 24시간이 지난 응답은 잠금 처리됩니다 (투명도 감소 및 잠금 아이콘 표시). 전체 AI 히스토리에 무제한으로 접근하려면 Pro로 업그레이드하세요.

## 트레이스 워터폴

트레이스 워터폴은 분산 호출 체인을 인터랙티브 다이어그램으로 시각화합니다.

### 워터폴 읽기

```
pipeline()                    ████████████████████████░░░░░ 450ms
├── embed()                   ██████████░░░░░░░░░░░░░░░░░░  180ms
├── search()                  ░░░░░░░░░░████████░░░░░░░░░░  150ms
└── format_result()           ░░░░░░░░░░░░░░░░░░██░░░░░░░░   20ms
```

- **수평 막대**는 스팬 지속 시간을 나타냅니다
- **중첩**은 부모-자식 관계를 보여줍니다
- **색상**은 상태를 나타냅니다:
  - 초록색 = 성공
  - 빨간색 = 에러
  - 파란색 = 캐시 히트
  - 회색 = 진행 중

### 스팬 상세 정보

스팬을 클릭하면 확인할 수 있는 정보:

- 함수 이름 및 모듈 경로
- 입력 파라미터 및 반환값
- 에러 상세 정보 및 스택 트레이스 (실패한 경우)
- 소요 시간 분석 및 캐시 히트 여부

### 트레이스 필터링

- 함수 이름, 상태, 소요 시간 범위로 필터링
- 타임스탬프, 소요 시간, 에러 횟수로 정렬
- Trace ID로 트레이스 검색

## AI Healer

AI Healer는 에러를 자동으로 진단하고 코드 수정을 제안합니다. OpenAI API 키가 필요합니다.

### 단건 진단

1. 드롭다운에서 함수 선택
2. **lookback** 슬라이더로 시간 범위 조절
3. **Diagnose & Heal** 클릭
4. 진단 결과 확인: 근본 원인 분석 및 코드 수정 제안 (구문 하이라이팅된 Python 코드)
5. 수정 코드를 복사하거나, 진단 결과를 AI History에 **Save** (Pro만 가능)

### 배치 진단

1. **Batch Mode**로 전환
2. 체크박스로 여러 함수 선택
3. **Batch Diagnose (N)** 클릭
4. 요약 뷰에서 모든 진단 결과 검토

### 시간 범위

lookback 범위 선택: 1시간, 3시간, 6시간, 12시간, 24시간, 3일, 7일, 또는 전체.

## 리플레이 테스트

AI 함수를 위한 시각적 회귀 테스트.

### 리플레이 실행

1. 사이드바에서 함수를 선택
2. 테스트 항목을 선택:
   - **Golden만** — 검증된 Golden Dataset 항목만
   - **전체** — 저장된 모든 실행
3. 비교 모드를 선택:
   - **정확 일치** — 문자 단위 비교
   - **시맨틱** — 설정 가능한 임계값을 사용한 임베딩 유사도
4. **리플레이 실행** 클릭

### 결과 보기

- **요약** — 통과/실패 건수, 성공률
- **상세 보기** — 예상 출력과 실제 출력의 나란히 비교
- **유사도 점수** — 시맨틱 비교의 코사인 유사도
- **Diff 보기** — 정확 일치 실패에 대한 문자 단위 diff

### Golden Dataset 관리

- 함수별 모든 Golden Dataset 항목 탐색
- 새로운 실행을 Golden 상태로 승격
- 오래된 항목 제거
- AI가 추천하는 승격 후보 확인

## 시맨틱 검색

모든 AI 함수 데이터에 대한 전문 및 벡터 검색.

### 검색 인터페이스

- 상단의 자연어 검색바
- 입력 시 실시간 결과 표시
- 함수 이름, 상태, 날짜 범위, 에러 코드로 필터링

### 검색 결과

결과에 포함되는 항목:

- **함수** — 매칭 함수 정의
- **실행** — 입출력 미리보기가 포함된 매칭 실행 로그
- **에러** — 스택 트레이스가 포함된 매칭 에러 로그
- **트레이스** — 워터폴 미리보기가 포함된 매칭 트레이스 체인

## 설정

### 외관

세 가지 테마 옵션: **Dark** (네온 다크 모드), **Light** (클린 라이트 모드), **System** (OS 설정 따라감).

### AI 설정

모든 AI 기능을 위한 글로벌 OpenAI API 키를 설정합니다.

### 연결

Weaviate 연결을 관리합니다. 연결별 OpenAI API 키를 설정하여 독립적인 AI 접근을 구성할 수 있습니다.

### 계정

프로필, 플랜 상태, 보안 설정을 확인합니다. 플랜을 업그레이드 또는 다운그레이드할 수 있습니다.
