## VectorSurferSTL이란?

**VectorSurferSTL**은 [VectorSurfer](/docs/vectorsurf-overview)의 경량 Python 전용 대안입니다. **Streamlit** 기반으로, 최소한의 설정으로 핵심 모니터링 기능을 제공합니다 — 프론트엔드 빌드 과정이 필요 없습니다.

| 기능 | VectorSurfer | VectorSurferSTL |
|---|---|---|
| **적합한 경우** | 프로덕션, 팀 | 빠른 세팅, 로컬 개발 |
| **설정 시간** | 대시보드 준비됨 | ~1분 |
| **AI Healer** | 전체 UI | 기본 UI |
| **다국어** | 영어/일본어/한국어 | 영어만 |
| **커스터마이징** | 높음 | 보통 |

## 빠른 시작

```bash
pip install vectorsurferstl
```

```bash
vectorsurferstl
```

브라우저에서 `http://localhost:8501`을 엽니다.

## 사전 요구사항

- **Python** 3.10+
- VectorWave 데이터가 있는 **Weaviate** 실행 중

이미 VectorWave를 위해 Weaviate를 실행하고 있다면, 추가 데이터베이스 설정은 필요 없습니다.

## 기능

### 대시보드

- 함수 실행 횟수 및 성공률
- 평균 지연시간 지표
- 캐시 히트율
- 에러 횟수 및 최근 에러

### 실행 브라우저

- 모든 추적된 함수 실행 목록
- 함수 이름, 상태, 날짜 범위로 필터링
- 실행 상세 정보 확인: 입력, 출력, 에러

### 검색

- 키워드로 실행 검색
- 함수, 팀, 상태로 필터링

### AI Healer (기본)

- 진단할 에러 선택
- AI가 생성한 근본 원인 분석 확인
- 코드 수정 제안 확인

## 어떤 것을 선택해야 할까?

**VectorSurferSTL을 선택하는 경우:**
- VectorWave 데이터를 빠르게 시각적으로 확인하고 싶을 때
- 로컬에서 개발 중이고 최소한의 설정을 원할 때
- 올 Python 스택을 선호할 때

**VectorSurfer를 선택하는 경우:**
- 프로덕션 환경에서 운영할 때
- 여러 팀원이 접근해야 할 때
- 고급 기능이 필요할 때 (트레이스 워터폴, 배치 치유, 리플레이 테스트)
