## 사전 요구사항

- **Node.js** 18+ (Next.js 프론트엔드용)
- **Python** 3.10+ (FastAPI 백엔드용)
- **Docker** (Weaviate용)
- **Git**

## 1. Weaviate 시작

VectorSurfer는 VectorWave와 동일한 Weaviate 인스턴스에 연결합니다. 이미 Weaviate가 실행 중이라면 이 단계를 건너뛰세요.

### Docker Compose (권장)

`vw_docker.yml`을 생성합니다:

```yaml
version: '3.4'
services:
  weaviate:
    image: semitechnologies/weaviate:latest
    ports:
      - "8080:8080"
      - "50051:50051"
    environment:
      ENABLE_MODULES: 'text2vec-openai'
      DEFAULT_VECTORIZER_MODULE: 'text2vec-openai'
      OPENAI_APIKEY: ${OPENAI_API_KEY}
      AUTHENTICATION_ANONYMOUS_ENABLED: 'true'
      PERSISTENCE_DATA_PATH: '/var/lib/weaviate'
      CLUSTER_HOSTNAME: 'node1'
    volumes:
      - ./weaviate-data:/var/lib/weaviate

  console:
    image: semitechnologies/weaviate-console:latest
    ports:
      - "8081:8080"
    environment:
      WEAVIATE_URL: "http://weaviate:8080"
```

```bash
docker-compose -f vw_docker.yml up -d
```

다음이 시작됩니다:
- **Weaviate** -- `http://localhost:8080`
- **Weaviate Console** -- `http://localhost:8081` (Weaviate 웹 UI, 선택사항)

## 2. VectorSurfer 클론

```bash
git clone https://github.com/Cozymori/VectorSurfer
cd VectorSurfer
```

## 3. 프론트엔드 설정

```bash
npm install
```

### 프론트엔드 환경 변수

프로젝트 루트에 `.env.local`을 생성합니다:

```bash
# Weaviate 연결
WEAVIATE_URL=http://localhost:8080

# 백엔드 API URL
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## 4. 백엔드 설정

```bash
cd backend
pip install -r requirements.txt
```

### 백엔드 환경 변수

`backend/` 디렉토리에 `.env`를 생성합니다:

```bash
# Weaviate
WEAVIATE_URL=http://localhost:8080

# OpenAI (AI Healer 및 RAG용)
OPENAI_API_KEY=sk-...

# GitHub (AI Healer를 통한 자동 PR용)
GITHUB_TOKEN=ghp_...
GITHUB_REPO=your-org/your-repo
```

## 5. 개발 서버 실행

### 방법 A: 동시 실행 (권장)

프로젝트 루트에서:

```bash
npm run dev
```

Next.js 프론트엔드와 FastAPI 백엔드가 동시에 시작됩니다.

### 방법 B: 별도 터미널

**터미널 1 -- 프론트엔드:**

```bash
npm run dev:frontend
# → http://localhost:3000
```

**터미널 2 -- 백엔드:**

```bash
cd backend
python run.py
# → http://localhost:8000
```

## 6. 확인

브라우저에서 `http://localhost:3000`을 엽니다. VectorSurfer 대시보드가 표시됩니다.

VectorWave가 이미 Weaviate에 데이터를 저장한 경우 대시보드에 다음이 표시됩니다:

- 실제 데이터가 포함된 KPI 카드
- 실행 타임라인
- 사이드바의 함수 목록

Weaviate가 비어 있으면 대시보드에 플레이스홀더 상태가 표시됩니다.

## 문제 해결

### 백엔드 연결 거부

```
Error: Failed to fetch from http://localhost:8000
```

FastAPI 백엔드가 실행 중인지 확인하세요:

```bash
cd backend && python run.py
```

### Weaviate 연결 실패

```
ConnectionError: Connection to Weaviate failed
```

Docker가 실행 중인지 확인하세요:

```bash
docker ps | grep weaviate
```

### 빈 대시보드

대시보드에 데이터가 표시되지 않으면, VectorWave로 실행 로그가 기록되었는지 확인하세요:

```python
from vectorwave import vectorize, initialize_database

initialize_database()

@vectorize(auto=True, capture_return_value=True)
def test_function(x: str):
    return f"Hello {x}"

test_function("World")
```

그런 다음 VectorSurfer 대시보드를 새로고침합니다.
