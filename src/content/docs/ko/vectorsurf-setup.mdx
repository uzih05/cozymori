## 사전 요구사항

VectorSurfer를 사용하기 전에 필요한 것:

- Python 프로젝트에 **VectorWave**가 설치 및 설정되어 있어야 합니다 ([설치](/docs/vectorwave-installation))
- VectorWave 데이터가 있는 **Weaviate** 인스턴스가 실행 중이어야 합니다
- 최신 **웹 브라우저**

## 1단계: 회원가입

VectorSurfer 대시보드를 방문하여 계정을 생성하세요:

**[VectorSurfer 열기 →](https://vectorsurfer.com)**

1. **Sign Up** 클릭
2. **이메일**과 **비밀번호** 입력
3. 선택적으로 **표시 이름** 추가
4. **Sign Up** 클릭으로 계정 생성

이미 계정이 있다면 **Log In**을 클릭하고 이메일과 비밀번호를 입력하세요.

## 2단계: Weaviate 연결 생성

로그인 후 **Projects** 페이지로 이동합니다. 여기서 VectorSurfer를 Weaviate 인스턴스에 연결합니다.

1. **Create Connection** 클릭
2. **연결 타입** 선택:

| 타입 | 설명 |
|---|---|
| **Self Hosted** | 직접 운영하는 Weaviate 인스턴스 (Docker, Kubernetes 등) |
| **WCS Cloud** | [Weaviate Cloud](https://console.weaviate.cloud/)의 관리형 인스턴스 |

3. 연결 정보 입력:

| 필드 | 기본값 | 설명 |
|---|---|---|
| **Name** | Default | 연결의 라벨 (예: "Production") |
| **Host** | `localhost` | Weaviate 서버 호스트명 또는 IP |
| **HTTP Port** | `8080` | Weaviate REST API 포트 |
| **gRPC Port** | `50051` | Weaviate gRPC 포트 |
| **API Key** | — | 인증 키 (WCS Cloud 필수, Self Hosted 선택사항) |

4. **Test Connection**을 클릭하여 연결 확인
5. 테스트 통과 시 **Create**를 클릭하여 연결 저장
6. 연결 카드에서 **Activate**를 클릭하여 활성화

> **처음이신가요?** Projects 페이지 하단에 빠른 시작 가이드가 나타납니다. **Start Guide**를 클릭하면 Weaviate 설정과 VectorWave 설치를 안내하는 단계별 가이드를 확인할 수 있습니다.

## 3단계: VectorWave 설정

아직 설정하지 않았다면, VectorWave를 설치하고 함수 실행 로깅을 시작하세요:

```bash
pip install vectorwave
```

```python
from vectorwave import vectorize, initialize_database

initialize_database()

@vectorize(auto=True, capture_return_value=True)
def your_function(prompt: str):
    # AI 함수 로직
    return result
```

`@vectorize`로 데코레이팅된 함수는 모든 실행이 자동으로 Weaviate에 저장됩니다. VectorSurfer는 동일한 Weaviate 인스턴스에서 읽어 데이터를 즉시 표시합니다.

자세한 설정 방법은 [VectorWave 빠른 시작](/docs/quickstart)을 참고하세요.

## 4단계: AI 기능 설정 (선택사항)

**Ask AI**, **Healer 진단**, **시맨틱 검색** 등 AI 기반 기능은 OpenAI API 키가 필요합니다.

### 글로벌 API 키

1. 사이드바에서 **Settings** (톱니바퀴 아이콘)으로 이동
2. **AI Settings** 탭 열기
3. OpenAI API 키 입력 (`sk-...`)
4. **Save** 클릭

이 키는 모든 연결의 모든 AI 기능에 사용됩니다.

### 연결별 API 키

특정 연결에 별도의 OpenAI 키를 설정할 수도 있습니다:

1. **Settings** → **Connections** 탭으로 이동
2. 연결 옆의 키 아이콘 클릭
3. API 키를 입력하고 저장

API 키가 설정된 연결은 카드에 **AI Ready** 배지가 표시됩니다.

> AI 기능은 선택사항입니다 — OpenAI 키 없이도 대시보드에서 모니터링, 트레이싱, 실행 조회가 가능합니다.

## 5단계: 대시보드 탐색

연결이 활성화되고 VectorWave가 실행을 기록하면 대시보드에 데이터가 표시됩니다:

- **KPI 카드** — 총 실행 수, 성공률, 에러율, 캐시 히트율, 평균 소요시간
- **실행 타임라인** — 시간별 함수 활동
- **최근 에러** — 최신 에러 한눈에 보기
- **시스템 상태** — Weaviate 연결 상태

대시보드는 완전히 커스터마이징 가능합니다 — **Edit** 버튼을 클릭하여 위젯을 추가, 제거, 크기 조절, 순서 변경할 수 있습니다. 사용 가능한 모든 위젯은 [사용 가이드](/docs/vectorsurf-features)를 참고하세요.
