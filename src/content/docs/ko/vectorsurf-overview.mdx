## VectorSurfer란?

**VectorSurfer**는 실시간 AI 관측 대시보드입니다. [VectorWave](/docs/vectorwave-overview) 데이터에 연결하여 AI 함수 호출을 모니터링, 추적, 치유하기 위한 시각적 도구를 제공합니다.

```
Your App + @vectorize → Weaviate ← VectorSurfer Dashboard
```

VectorWave가 함수 실행을 캡처하면, VectorSurfer가 이 데이터를 실행 가능한 인사이트로 변환합니다 — 하나의 통합 인터페이스에서.

## 주요 기능

### 실시간 대시보드

모든 핵심 지표를 한눈에 보여주는 반응형 위젯 그리드 대시보드:

- **KPI 카드** — 총 호출 수, 성공률, 에러 수, 캐시 히트율, 평균 소요시간
- **실행 타임라인** — 성공/에러/캐시 분류가 적용된 시간별 함수 실행 현황
- **토큰 사용량** — 카테고리별 LLM 토큰 소비량
- **시스템 상태** — Weaviate 연결 상태 및 등록된 함수 수

### 트레이스 워터폴

분산 트레이스를 인터랙티브 워터폴 다이어그램으로 시각화:

- 데코레이팅된 함수의 전체 호출 체인 확인
- 병목 구간과 느린 스팬을 즉시 파악
- 스팬을 클릭하면 상세 실행 데이터 확인
- 상태별 색상 구분 (성공, 에러, 캐시됨)

### AI Healer

GPT-4 기반 에러 진단을 시각적 인터페이스로 제공:

- 전체 컨텍스트가 포함된 최근 에러 탐색
- 원클릭 근본 원인 분석
- diff 하이라이팅이 적용된 생성 코드 수정 확인
- 대시보드에서 직접 GitHub PR 생성
- 여러 에러에 대한 배치 치유

### 리플레이 테스트

AI 함수를 위한 시각적 회귀 테스트:

- 함수 및 Golden Dataset 항목 선택
- 실시간 진행 상황과 함께 리플레이 실행
- 예상 출력과 실제 출력을 나란히 비교
- 정확 일치 및 시맨틱 비교 모두 지원

### Ask AI

RAG 기반 VectorWave 데이터에 대한 자연어 Q&A:

- 함수, 에러, 실행 패턴에 대해 질문
- AI가 Weaviate에서 관련 컨텍스트를 검색한 후 답변 생성
- 응답은 자동으로 AI 히스토리에 저장

### AI 히스토리 (저장된 응답)

모든 AI 생성 응답을 탐색하고 관리:

- 소스 (Ask AI / Healer), 북마크 상태, 키워드로 필터링
- 중요한 응답을 북마크하여 빠르게 접근
- Free 플랜: 일 5회 AI 호출, 24시간 후 응답 잠금
- Pro 플랜: 무제한 호출 및 전체 히스토리 접근

### 시맨틱 검색

모든 AI 함수 데이터에 대한 자연어 검색:

- 함수, 실행 로그, 에러 검색
- 하이브리드 검색 (벡터 유사도 + 키워드 필터링)
- RAG 기반 Q&A로 코드베이스 탐색

### 다국어 UI

영어, 일본어, 한국어 UI를 기본 제공하며, 글로벌 팀을 위해 설계되었습니다.

## 동작 방식

VectorSurfer는 VectorWave가 데이터를 쓰는 동일한 Weaviate 인스턴스에서 데이터를 읽습니다. 추가 설정이 필요 없습니다 — Python 코드에서 `@vectorize`만 사용하면 VectorSurfer가 모든 것을 보여줍니다.

VectorSurfer를 사용하기 위해 VectorWave가 반드시 필요한 것은 아니지만, SDK가 대시보드에 표시되는 데이터를 생성합니다.
