## VectorSurfer란?

**VectorSurfer**는 [VectorWave](/docs/vectorwave-overview) 데이터를 실시간으로 모니터링하기 위한 풀스택 관측 대시보드입니다. **Next.js 16 + FastAPI** 기반으로, VectorWave가 캡처하는 모든 데이터에 대한 시각적 도구를 제공합니다.

```
┌─────────────────────────────────────────┐
│            VectorSurfer                  │
│                                          │
│  ┌──────────┐       ┌───────────────┐   │
│  │ Next.js  │ ←───→ │   FastAPI     │   │
│  │ Frontend │       │   Backend     │   │
│  │          │       │               │   │
│  │ • UI     │       │ • Weaviate    │   │
│  │ • Charts │       │ • OpenAI API  │   │
│  │ • i18n   │       │ • PyGithub    │   │
│  └──────────┘       └───────────────┘   │
│                          │               │
│                     Weaviate             │
│               (VectorWave와 공유)        │
└─────────────────────────────────────────┘
```

## 주요 기능

### 벤토 그리드 대시보드

드래그 앤 드롭 반응형 대시보드와 위젯:

- **KPI 카드** -- 총 호출 수, 성공률, 평균 지연시간, 캐시 히트율
- **실행 타임라인** -- 시간별 함수 실행 현황 실시간 차트
- **에러 분포** -- 에러 유형 및 빈도 분석
- **드리프트 모니터** -- Golden Dataset 중심점으로부터의 시각적 거리

### 트레이스 워터폴

분산 트레이스를 인터랙티브 워터폴 다이어그램으로 시각화:

- `@vectorize` + `@trace_span` 함수의 전체 호출 체인 확인
- 병목 구간과 느린 스팬을 한눈에 파악
- 스팬을 클릭하면 상세 실행 데이터 확인
- 상태별 색상 구분 (성공, 에러, 캐시됨)

### AI Healer

GPT-4 기반 에러 진단을 시각적 인터페이스로 제공:

- 전체 컨텍스트가 포함된 최근 에러 탐색
- [VectorWaveHealer](/docs/vectorwave-healing)를 통한 원클릭 진단
- diff 하이라이팅이 적용된 생성 코드 수정 확인
- 대시보드에서 직접 GitHub PR 생성
- 여러 에러에 대한 배치 치유

### 리플레이 테스트

시각적 리플레이 테스트 UI:

- 함수 및 Golden Dataset 항목 선택
- 실시간 진행 상황과 함께 리플레이 실행
- 예상 출력과 실제 출력을 나란히 비교
- 정확 일치 및 [시맨틱 비교](/docs/vectorwave-testing) 모두 지원

### 시맨틱 검색

모든 VectorWave 데이터에 대한 자연어 검색:

- 함수, 실행 로그, 에러 검색
- 하이브리드 검색 (벡터 유사도 + 키워드 필터링)
- 전체 실행 상세 정보로 클릭하여 이동
- [RAG Q&A](/docs/vectorwave-search)와 통합

### 다국어 UI

영어, 일본어, 한국어 UI를 기본 제공하며, 글로벌 팀을 위해 설계되었습니다.

## VectorWave와의 관계

VectorSurfer는 VectorWave의 **동반 대시보드**입니다. VectorWave가 데이터를 쓰는 동일한 Weaviate 인스턴스에서 데이터를 읽습니다:

```
Your App + @vectorize → Weaviate ← VectorSurfer Dashboard
```

VectorWave를 사용하기 위해 VectorSurfer가 반드시 필요한 것은 아닙니다 -- SDK는 독립적으로 동작합니다. 하지만 VectorSurfer를 사용하면 다음이 훨씬 쉬워집니다:

- 함수 상태를 실시간으로 모니터링
- 에러를 시각적으로 진단
- Golden Dataset 관리
- 리플레이 테스트 실행

## 대안 비교

| 대시보드 | 유형 | 적합한 경우 |
|---|---|---|
